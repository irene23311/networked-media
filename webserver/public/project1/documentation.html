Documentation

Version: 1.0.0

Concept:
A digital representation of a physical desktop.
This project is a collage-based hypertext website that presents objects on my desk as entry points.
Each object links to its own page, where fragments of text describe use, habit, or association with myself Navigation through clicking is the primary way the story unfolds.

Usage:
Click items to inspect closer.
Use the "Map" for a linear view.

Process

I started by sketching a rough layout of objects on a desk and deciding which items felt representative. I then built a simple HTML structure with an index page and individual item pages.
